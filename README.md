# ChoreoLab

ChoreoLab is a simple prototype of a choreography visualization tool built with React. The project has been cleaned up so that all source files now live at the repository root.

## Features
- Upload JSON pose data and audio files
- Visualize multiple dancers
- Drag-and-drop dancers on a grid
- Save and label formations
- Animated transitions and timeline syncing

## Development
Install dependencies and start the development server:
```bash
npm install
npm start
```

To create a production build run:
```bash
npm run build
```

## Repository layout
- `src/` – React components and styles
- `public/` – static assets and the main `index.html`
- `package.json` – project metadata and scripts

The repository includes a `.gitignore` to keep `node_modules/` and build artifacts out of version control.
